# Changelog

All notable changes to this portfolio are documented here.

## [1.0.0] - 2026-04-04

### Added

- Multi-page React + Vite portfolio shell
- GitHub-backed project ingestion and sorting
- Project detail pages and demo pages
- Express backend for analytics
- Analytics dashboard and tracking endpoints
- Contact form backend with Nodemailer and Gmail SMTP
- Vercel Analytics integration
- Deployment config for Vercel frontend routing
- Build-time sitemap and robots generation
- Centralized site config
- Branded SVG logo and favicon

### Changed

- Replaced legacy username references with `theworker02`
- Refined card centering and shared hover behavior
- Updated contact copy and real backend submission flow
- Simplified the app by removing the snapshot/version UI after experimentation
- Cleaned navbar and app shell behavior for a smoother production setup

### Removed

- Snapshot/version switcher UI
- Snapshot-specific routes and related components
- Unused version banner and snapshot styling
